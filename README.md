# RVR Analysis Project

## 项目背景

机场 RVR（Runway Visual Range）设备存在间歇性数据丢失的问题。

本项目旨在利用 Python、Pandas、SQL 对历史数据进行分析，
定位故障原因，并逐步实现自动化故障诊断工具。

## 2.数据审视

### 审视内容：

- 文件路径检查
- 文件基本信息获取
- 文件预览（前10行）
- 分隔符识别

### 遇到的问题

### 遇到的问题及解决方案

| 问题              | 解决方案                                  |
| ----------------- | ----------------------------------------- |
| Notebook 模块缓存 | 重新加载模块或重启 Kernel                 |
| 函数调用路径      | 使用 `sys.path.append()` 添加 `src` 路径  |
| Git 双仓库        | 保留两个仓库，分别管理学习项目和 RVR 项目 |
| 文件编码未知      | 暂时使用 `latin1`，后续实现自动检测       |
